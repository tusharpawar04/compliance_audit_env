WHAT WAS WRONG
=================================================================

The environment had 3 critical OpenEnv protocol violations:

1. step() returned tuple instead of Observation
   - OpenEnv expects: def step(action) -> Observation
   - We had: def step(action) -> tuple[Observation, float, bool, dict]
   - Fix: Return only Observation with reward/done embedded

2. WebSocket messages missing 'data' wrapper
   - OpenEnv expects: {'type': 'reset', 'data': {'task': 'easy'}}
   - We sent: {'type': 'reset', 'task': 'easy'}
   - Fix: Nest all parameters under 'data' key

3. WebSocket connection check used wrong attribute
   - websockets uses: .close_code is not None
   - We used: .closed (doesn't exist)
   - Fix: Use correct attribute
